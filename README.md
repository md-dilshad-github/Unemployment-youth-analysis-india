# Unemployment-youth-analysis-india
Unemployment trends and youth labour market analysis in India using Excel and Python.

## Objective: Practice Analysing Unemployment trends and to understand trend patterns, demographic differences, and economic implications using Excel and Python.

Data Source: Periodic Labour Force Survey (PLFS) – Key Indicators (Government of India)

## Tools Used:
- Microsoft Excel (data cleaning and visualisation)
- Python (pandas, matplotlib)
  
## Methodology
- Created a cleaned dataset from PLFS key indicators.
- Visualised unemployment trends and youth unemployment patterns.
- Conducted basic correlation analysis between overall and youth unemployment.

## Key Insights:
- Youth unemployment consistently exceeds overall unemployment.
- Urban unemployment shows higher volatility than rural unemployment.
- Youth employment remains vulnerable during economic slowdowns.

## Possible Outcome:
The analysis highlights structural challenges in India’s youth labour market and the importance of targeted policy and skill interventions.
